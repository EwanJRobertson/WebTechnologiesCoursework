Cave - The Game

About

Cave is a browser-based choose your own adventure game, created for my Web Technologies coursework. The goal of the game is to escape the cave, that the player has found themselves in, with data from a terminal that sits in the cave. The choices you make throughout the game affect your future choices and the overall outcome of the story.

Instructions

The game is split into two areas. The "game area" where the pictures are display, and the "messages" where any decisions and text is displayed.

Images

Images displayed in the game area may have an event attached to them that activates when the image is clicked. These images will gain an orange outline when hovered over or when the "highlight" button is clicked. The "highlight" button is represented by a magnifying glass and sits in the top right corner of the game area.

Text Input

Text Input is handled by a text box and submit button. You are required to input at least 1 character into a text box. A red outline will appear around the element if an invalid value is entered.

Choices

Player choices are displayed as a stacked group of buttons. If a choice has (PER), (STR) or (LUC) at the start of the text then it is only being displayed because of the players primary attribute, a value set near the start of the game.

100s

Timer

At a certain point in the game there is a timer that starts. This is only visible if the player has made certain choices up to this point. If the timer reaches 0, whether it is being displayed or not, the choice will be taken away from the player.
